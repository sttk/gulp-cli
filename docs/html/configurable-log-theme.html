Configurable Log Theme

Gulp's log outputs have theme and the theme can be configured with .gulp.* file. Since this function is realized by using theming-log, the format of log message templates follows the specification of it. An example is as below:

// .gulp.js
module.exports = {
  log: {
    theme: {
      ...,
      TASK: '{cyan: {1}}',
      TASKS: {
        NAME: '{TASK: {1}}',
        ...,
      },
      ...,
    },
  },
};
