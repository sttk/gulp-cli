Configurable Log Messages

Gulp's log messages can be configured with .gulp.* file. Since this function is realized by using theming-log, the format of log message templates follows the specification of it. An example is as below:

// .gulp.js
module.exports = {
  log: {
    messages: {
      help: {
        usage: '\n{TITLE: Usage:} gulp {OPTION: [options]} {TASK: tasks}',
        ...,
      },
      ...,
    },
  },
};
